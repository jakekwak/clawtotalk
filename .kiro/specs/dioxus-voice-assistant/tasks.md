# 구현 계획: Dioxus Voice Assistant

## 개요

ClawToTalk 웹 애플리케이션을 Dioxus 0.7을 사용하여 크로스 플랫폼 네이티브 애플리케이션으로 변환하는 구현 계획입니다.

**아키텍처**: 서버-클라이언트 구조로, 클라이언트 앱은 Mac Mini 또는 VPS에서 실행되는 OpenClaw 서버와 HTTP REST API로 통신합니다.
